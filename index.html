00 / POSITIONING

Breadth is useful when it closes a real gap.

My background is not a list of unrelated technologies. Network management taught operational constraints. Radar, camera, and FPGA work taught physical constraints. Access-control platforms taught device, identity, event, and enterprise boundaries. AI-native work now turns that accumulated context into developer platforms and verification systems.

제 경력은 서로 무관한 기술의 목록이 아닙니다. 네트워크 관리에서 운영 제약을, 레이더·카메라·FPGA 개발에서 물리 제약을, 출입통제 플랫폼에서 장치·신원·이벤트·엔터프라이즈 경계를 배웠습니다. 현재는 이 맥락을 AI-native 개발자 플랫폼과 검증 시스템으로 전환하고 있습니다.
