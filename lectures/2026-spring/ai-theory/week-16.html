Week 16

2026-06-15 - 06-21 · FOL, Inference, and Final Assessment

First-Order Logic

교안: 09-1. FOL.pdf, 09-2. FOL Inference.pdf (AIMA 8-9장) · 핵심 개념 중심의 개인 요약

16주차는 명제논리보다 표현력이 높은 1차 논리(First-Order Logic, FOL)와 그 추론 방법을 다루며 학기를 마무리했다. 객체·관계·함수를 표현하는 언어에서 출발해 양화사, 예화, Skolem 상수, 단일화, Generalized Modus Ponens로 이어졌다.

핵심 관점. 명제논리가 각각의 사실을 독립 기호로 다룬다면, FOL은 세계를 객체와 객체 사이의 관계로 구조화한다. 하나의 일반 규칙을 여러 객체에 적용할 수 있어 더 압축적이고 재사용 가능한 지식 표현이 된다.

1. 왜 FOL이 필요한가

Objects

사람, 장소, 숫자, 사물처럼 세계를 구성하는 대상이다.

Relations

대상의 속성이나 대상 사이의 연결을 참·거짓으로 표현한다.

Functions

하나 이상의 대상을 입력받아 다른 대상을 가리키는 값을 만든다.

명제논리로 일반 법칙을 표현하려면 모든 개별 경우를 따로 작성해야 한다. FOL은 변수와 양화사를 사용해 대상 전체에 적용되는 규칙이나 어떤 대상의 존재를 하나의 문장으로 나타낸다.

2. FOL 언어의 구성

요소 역할 예
Constant 특정 객체를 가리킨다. John, SKKU, 2
Variable 양화되거나 치환될 객체 자리를 나타낸다. x, y
Predicate 속성이나 관계를 참·거짓 문장으로 만든다. Student(x), Knows(x, y)
Function term을 입력받아 다른 객체를 가리키는 term을 만든다. MotherOf(x)
Equality 두 term이 같은 객체를 가리키는지 표현한다. FatherOf(John) = Henry
Term = constant | variable | function(term, ...)
Atomic sentence = predicate(term, ...) | term = term

3. 양화사와 범위

양화사 의미 전형적인 문장 구조
Universal 모든 객체에 대해 문장이 참이다. 조건을 나타내는 implication과 자주 결합한다.
Existential 문장을 만족하는 객체가 적어도 하나 존재한다. 속성을 함께 요구하는 conjunction과 자주 결합한다.

같은 종류의 양화사는 순서를 바꾸어도 의미가 유지되지만, universal과 existential의 순서를 섞으면 뜻이 달라질 수 있다. 또한 두 양화사는 부정과 함께 서로 바꿔 표현할 수 있다.

주의. 모든 학생에 대한 규칙을 쓸 때 universal 뒤에 단순 conjunction을 두면 모두가 그 조건을 실제로 만족한다는 과도한 주장이 될 수 있다. 반대로 existential 뒤에 implication만 두면 조건을 만족하지 않는 객체 때문에 문장이 쉽게 참이 될 수 있다.

4. 해석과 지식 표현

FOL 모델은 객체들의 집합과 함께 각 상수, 술어, 함수가 무엇을 가리키는지 정하는 interpretation을 포함한다. 문장의 참·거짓은 기호 모양만으로 정해지는 것이 아니라 이 해석에 따라 결정된다.

Atomic facts

특정 객체의 속성이나 두 객체의 관계를 표현한다.

General rules

변수와 양화사로 여러 개별 사실을 하나의 규칙으로 묶는다.

Structured meaning

시간, 위치, 행동, 자연어 의미를 객체와 관계로 연결한다.

5. FOL 추론: 예화에서 단일화까지

방법 핵심 동작 의미
Universal Instantiation universal 변수에 ground term을 치환한다. 일반 규칙에서 필요한 개별 문장을 만든다.
Existential Instantiation existential 변수를 새 Skolem 상수로 한 번 치환한다. 어떤 객체가 존재한다는 사실을 새 이름으로 다룬다.
Unification 두 표현을 같게 만드는 변수 치환을 찾는다. 규칙의 전제와 알려진 사실을 연결한다.
Generalized Modus Ponens 단일화된 전제들과 규칙에서 치환된 결론을 만든다. 불필요한 모든 ground 문장을 만들지 않고 직접 추론한다.

모든 universal 문장을 가능한 ground term으로 바꾸는 명제화는 원리상 가능하지만 무관한 문장을 많이 만들 수 있다. 단일화와 GMP를 사용한 lifted inference는 현재 사실과 맞는 치환만 찾아 관련 결론을 직접 도출한다.

6. 주차 요약

  • FOL은 객체, 관계, 함수를 이용해 세계의 구조를 표현한다.
  • 상수, 변수, 술어, 함수, 등호, 연결사, 양화사가 FOL 문장을 구성한다.
  • Universal은 전체를, existential은 적어도 하나의 존재를 표현한다.
  • UI와 EI는 양화 문장을 개별 문장으로 예화하며, EI는 새 Skolem 상수를 사용한다.
  • Unification과 GMP는 관련 사실과 규칙을 직접 연결하는 lifted inference를 만든다.

Week 16 studied First-Order Logic (FOL), a language more expressive than propositional logic, and concluded the semester. The conceptual path moved from objects, relations, and functions to quantifiers, instantiation, Skolem constants, unification, and Generalized Modus Ponens.

Core perspective. Propositional logic treats each fact as a separate symbol. FOL structures a world through objects and relations among them. A single general rule can therefore apply across many objects, producing more compact and reusable knowledge representation.

1. Why FOL?

Objects

Entities such as people, places, numbers, and physical things.

Relations

Truth-valued properties of objects or links between objects.

Functions

Mappings from one or more objects to another object.

In propositional logic, a general law may require a separate sentence for every case. FOL uses variables and quantifiers to state a rule for all objects or assert that an object with certain properties exists.

2. Components of the FOL Language

Element Role Example
Constant Names a particular object. John, SKKU, 2
Variable Marks an object position to quantify or substitute. x, y
Predicate Turns a property or relation into a truth-valued sentence. Student(x), Knows(x, y)
Function Maps terms to a term that refers to another object. MotherOf(x)
Equality States that two terms refer to the same object. FatherOf(John) = Henry
Term = constant | variable | function(term, ...)
Atomic sentence = predicate(term, ...) | term = term

3. Quantifiers and Scope

Quantifier Meaning Typical sentence structure
Universal The sentence holds for every object. Often combines a condition and result through implication.
Existential At least one object satisfies the sentence. Often joins required properties through conjunction.

Quantifiers of the same type can exchange order without changing meaning, but mixing universal and existential order can change the claim. Negation also allows either quantifier to be expressed through the other.

Caution. A universal statement with a simple conjunction may accidentally claim that every object satisfies the condition. An existential statement with implication may become true merely because some object fails the premise.

4. Interpretation and Knowledge Representation

An FOL model contains a domain of objects and an interpretation that assigns meaning to every constant, predicate, and function. A sentence's truth is determined through that interpretation, not by symbol shape alone.

Atomic facts

Describe an object's property or a relation between particular objects.

General rules

Use variables and quantifiers to replace many separate facts with one rule.

Structured meaning

Connect time, location, action, and language meaning through objects and relations.

5. FOL Inference: From Instantiation to Unification

Method Core operation Purpose
Universal Instantiation Substitutes a ground term for a universal variable. Creates a needed instance of a general rule.
Existential Instantiation Replaces an existential variable once with a fresh Skolem constant. Gives a new name to an object known to exist.
Unification Finds a substitution that makes two expressions identical. Matches known facts with the premises of a rule.
Generalized Modus Ponens Uses unified premises and a rule to derive a substituted conclusion. Infers directly without generating every possible ground sentence.

Propositionalizing every universal sentence over available ground terms is possible in principle, but it can generate many irrelevant sentences. Lifted inference with unification and GMP instead finds substitutions that match current facts and derives only relevant conclusions.

6. Week Summary

  • FOL represents world structure through objects, relations, and functions.
  • Constants, variables, predicates, functions, equality, connectives, and quantifiers form FOL sentences.
  • Universal quantification covers all objects; existential quantification asserts at least one.
  • UI and EI instantiate quantified sentences, with EI introducing a fresh Skolem constant.
  • Unification and GMP support lifted inference that connects relevant facts and rules directly.
