Week 2

2026-03-09 · Systems of Linear Equations

Solutions · Elimination · Models

정리 기준: 02. 선형방정식 · 해의 구조, 소거법, 응용을 중심으로 재구성

연립선형방정식은 여러 조건을 동시에 만족하는 값을 찾는 문제다. 핵심은 계산 결과 하나보다 해가 유일한지, 존재하지 않는지, 자유변수 때문에 무수히 많은지를 행렬 구조에서 판별하는 것이다.

1. 해집합의 세 형태를 기하와 대수로 읽는다

  • 독립적인 제약들이 한 점에서 만나면 유일해가 존재하고 각 미지수를 결정하는 추축이 생긴다.
  • 같은 왼쪽 계수에서 서로 모순되는 상수가 나타나면 해가 없는 불능 시스템이다.
  • 어떤 방정식이 다른 방정식의 선형결합이면 자유변수가 남아 무수히 많은 해를 갖는 부정 시스템이 된다.

2. 동치인 행 연산으로 구조를 보존한다

행 연산의미주의점
두 행 교환방정식의 순서만 바꾼다.해집합은 변하지 않는다.
0이 아닌 상수배같은 제약을 다른 척도로 쓴다.0배는 정보를 지운다.
한 행의 배수를 다른 행에 더함미지수를 소거한다.원래 관계와 동치임을 유지한다.

3. 가우스 소거는 해의 모양을 드러내는 절차다

계수와 상수를 첨가행렬로 묶고 추축 아래를 소거하면 행 사다리꼴을, 추축 위까지 정리하면 기약행 사다리꼴을 얻는다. 추축변수와 자유변수를 구분하고 모순행을 확인하면 과소·과대결정 시스템도 같은 틀에서 해석할 수 있다.

Ax = b → 첨가행렬 [A | b] → REF → RREF → 추축·자유변수·모순행 판정

응용 연결. 다항식 곡선 맞춤은 선택한 점을 통과하도록 계수를 구하는 선형 시스템이고, 네트워크 흐름은 각 지점의 유입과 유출 제약을 동시에 만족시키는 시스템이다.

핵심 정리

  • 해의 개수는 방정식 수만으로 결정되지 않고 추축, 자유변수, 모순행의 관계로 결정된다.
  • 기본 행 연산은 해집합을 바꾸지 않으므로 복잡한 시스템을 해석하기 쉬운 동치 형태로 바꾼다.
  • 가우스-조단 소거는 숫자를 줄이는 계산이면서 계수·독립성·해공간을 읽는 구조 분석이다.

실전 팁

  • 소거 전에 변수 순서와 첨가열을 표시하고 모든 행 연산을 짧은 기호로 옆에 남긴다.
  • RREF를 얻은 뒤 곧바로 답을 쓰지 말고 추축열, 자유변수, 모순행을 세 번 따로 확인한다.
  • 구한 해를 원래 방정식에 대입해 검산하고, 자유변수가 있다면 매개변수형 해로 표현한다.

Basis: 02. Linear Equations · Reorganized around solution structure, elimination, and applications

A system of linear equations asks for values satisfying several constraints simultaneously. The main goal is not merely to produce numbers but to determine whether the solution is unique, impossible, or infinite because free variables remain.

1. Read the Three Solution Types Geometrically and Algebraically

  • Independent constraints meeting at one point produce a unique solution, with pivots determining every variable.
  • Conflicting constants attached to the same zero coefficient row reveal an inconsistent system with no solution.
  • If one equation is a linear combination of others, a free variable remains and the system admits infinitely many solutions.

2. Equivalent Row Operations Preserve the Structure

Row operationMeaningCaution
Swap two rowsReorder the equations.The solution set is unchanged.
Scale by a nonzero constantWrite the same constraint at another scale.Multiplication by zero destroys information.
Add a multiple of one row to anotherEliminate a variable.The new equation remains equivalent.

3. Gaussian Elimination Exposes the Shape of the Solution

Place coefficients and constants in an augmented matrix. Eliminating entries below each pivot gives row echelon form; clearing above pivots gives reduced row echelon form. Pivots, free columns, and inconsistent rows classify underdetermined and overdetermined systems within the same procedure.

Ax = b → augmented matrix [A | b] → REF → RREF → classify pivots, free variables, and contradictions

Application connection. Polynomial curve fitting solves for coefficients that make a curve pass through selected observations. Network analysis solves flow constraints that balance incoming and outgoing quantities at connected nodes.

Key Takeaways

  • The number of equations alone does not determine the number of solutions; pivots, free variables, and contradictions do.
  • Elementary row operations preserve the solution set while converting a difficult system into an interpretable equivalent form.
  • Gauss-Jordan elimination is both a computational method and a structural analysis of rank, independence, and the solution space.

Practical Tips

  • Fix the variable order and mark the augmented column before elimination; annotate each row operation in compact notation.
  • After reaching RREF, inspect pivot columns, free variables, and inconsistent rows separately before writing a solution.
  • Substitute the result into the original equations, and express infinite solutions parametrically rather than listing isolated examples.
