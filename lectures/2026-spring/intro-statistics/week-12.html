Confidence Intervals

Keywords: 신뢰구간, 표본오차, 표준오차, 신뢰계수, 유의수준, Z critical value, t distribution, 독립표본, 대응표본, 자유도, 카이제곱분포, 모분산, 가설검정

12주차는 신뢰구간을 중심으로 추정과 검정의 여러 케이스를 정리한 시간이었다. 핵심은 공식을 많이 외우는 것이 아니라, 하나의 기본 구조를 이해한 뒤 상황별로 표준오차와 신뢰계수를 바꾸어 끼워 넣는 것이다. 수업에서는 단일 평균, 두 모평균 차이, 독립표본과 대응표본, 모분산을 알 때와 모를 때, t분포와 카이제곱분포까지 연결했다.

핵심. 목표값은 추정값을 중심으로 표본오차만큼 흔들린다. 표본오차는 신뢰계수와 표준오차의 곱으로 생각하면 된다. 그래서 통계 방법론에서는 표준오차를 조금이라도 줄이는 일이 매우 중요하다.

1. 신뢰구간의 기본 공식

이번 주차에서 가장 중요한 뼈대는 다음 두 줄이다. 어떤 모수를 추정하든, 먼저 추정값을 놓고 그 주변에 표본오차를 붙여 목표값이 들어갈 만한 구간을 만든다.

목표값 = 추정값 +- 표본오차
표본오차 = 신뢰계수 x 표준오차
개념 의미
추정값 표본에서 계산한 값. 예: 표본평균, 표본비율, 두 표본평균의 차이
표준오차 추정값의 표준편차. 표본을 다시 뽑을 때 추정값이 얼마나 흔들리는지 나타낸다.
신뢰계수 정규분포, t분포, 카이제곱분포 등 해당 상황의 분포에서 얻는 임계값
표본오차 신뢰구간의 반폭. 신뢰계수와 표준오차를 곱해 얻는다.

표준오차의 "standard"는 추정값의 표준편차라는 뜻이다. 같은 모집단을 대상으로 표본을 여러 번 뽑으면 표본평균이나 표본비율 같은 추정값이 조금씩 달라진다. 그 추정값들의 흔들림을 표준편차로 표현한 것이 표준오차다.

2. 유의수준과 신뢰구간의 의미

유의수준은 보통 alpha로 적고, 아무 언급이 없으면 5%, 즉 alpha = 0.05를 기본값으로 둔다. 신뢰수준은 1 - alpha이고, 95% 신뢰구간은 alpha = 0.05일 때의 구간이다.

P(|오차| <= c) = 1 - alpha
P(theta_hat - c <= theta <= theta_hat + c) = 1 - alpha
[theta_hat - c, theta_hat + c] = theta에 대한 100(1 - alpha)% 신뢰구간

신뢰도가 높아질수록 신뢰구간의 폭은 넓어진다. 더 높은 확률로 모수를 포함하고 싶다면, 더 넓은 구간을 잡아야 한다. 따라서 신뢰구간을 읽을 때는 "신뢰수준"과 "구간의 폭"을 함께 봐야 한다.

3. Z 신뢰구간과 중요한 임계값

모분산 또는 모표준편차를 알고 있고 정규근사가 가능한 경우, 표준정규분포의 임계값을 사용한다. 가운데 확률이 1 - alpha가 되도록 양끝을 각각 alpha / 2만큼 잘라낸다.

P(-z_(alpha/2) < Z < z_(alpha/2)) = 1 - alpha
Z = (X_bar - mu) / (sigma / sqrt(n))
mu = x_bar +- z_(alpha/2) x (sigma / sqrt(n))
신뢰수준 alpha z_(alpha/2) 수업 메모
99% 0.01 2.58 가장 넓은 구간
95% 0.05 1.96 기본값처럼 자주 사용
90% 0.10 1.645 상대적으로 좁은 구간

경험적 법칙에서는 평균에서 1시그마 안에 약 68%, 2시그마 안에 약 95%, 3시그마 안에 약 99.7%가 들어간다고 말한다. 하지만 표준정규분포에서 정확히 95%를 가운데에 두려면 2가 아니라 약 1.96을 써야 한다. 그래서 95% 신뢰구간에서 z_(alpha/2)는 1.96이다.

4. 두 모평균 차이: 독립표본

두 집단의 평균 차이를 볼 때는 표본평균의 차이가 추정값이 된다. 독립표본이고 두 모분산을 알고 있다면 표본평균 차이의 분포는 다음과 같이 정리된다.

(X_bar - Y_bar) ~ N(mu_1 - mu_2, sigma_1^2 / n_1 + sigma_2^2 / n_2)
(mu_1 - mu_2) = (x_bar - y_bar) +- z_(alpha/2) x sqrt(sigma_1^2 / n_1 + sigma_2^2 / n_2)

모분산을 모를 때는 표본분산을 사용하고 t분포로 넘어간다. 두 모분산이 같다고 가정할 수 있으면 합동분산을 쓰고, 같다고 보기 어렵다면 Welch 방식의 자유도를 계산한다. 실제 계산에서는 자유도 값이 복잡하므로 컴퓨터가 계산하는 경우가 많다.

v = (s_1^2 / n_1 + s_2^2 / n_2)^2 / { (s_1^2 / n_1)^2 / (n_1 - 1) + (s_2^2 / n_2)^2 / (n_2 - 1) }
(mu_1 - mu_2) = (x_bar - y_bar) +- t_(alpha/2, v) x sqrt(s_1^2 / n_1 + s_2^2 / n_2)

5. 대응표본

쌍으로 데이터를 모으는 것을 대응표본이라고 한다. 예를 들어 같은 사람의 안정 시 심박수와 100미터 전력질주 후 심박수, 공복혈당과 식후 2시간 혈당처럼 한 대상에서 두 조건의 값을 같이 얻는 경우다.

구분 예시 핵심 처리
독립표본 서로 다른 두 집단의 평균 비교 두 표본평균 차이의 표준오차를 계산한다
대응표본 같은 대상의 전후 비교 각 쌍의 차이 d_i를 만든 뒤 단일평균 문제처럼 본다
d_i = x_i - y_i
mu_D = d_bar +- t_(alpha/2, n-1) x s_D / sqrt(n)

같은 데이터를 보더라도 관점에 따라 해석은 달라질 수 있다. 중간고사와 기말고사 점수 차이를 볼 때도 학생은 "중간고사가 어려웠다"고 느낄 수 있고, 교수는 "기말에 공부를 덜 했다"고 해석할 수 있다. 통계는 숫자를 계산하는 일이면서 동시에 어떤 비교 구조를 잡을지 판단하는 일이다.

6. 카이제곱분포와 모분산

표준정규분포를 따른다는 것은 Z = (X - mu) / sigma처럼 표준화할 수 있다는 뜻이다. 표본평균을 사용해 중심을 잡으면 자유도 하나가 줄어들고, 제곱합은 카이제곱분포를 따른다.

sum((X_i - X_bar)^2 / sigma^2) ~ chi-square_(n-1)
sum(((X_i - X_bar) / sigma)^2) ~ chi-square_(n-1)

X_bar는 모수가 아니라 통계량이고, 모평균을 추정하는 값이다. 표본평균을 사용했기 때문에 자유도는 n - 1로 표기한다. 모분산에 대한 신뢰구간은 이런 카이제곱분포의 임계값을 사용한다.

7. 케이스를 외우지 말고 구조로 추론하기

신뢰구간은 단일평균, 모분산을 알 때와 모를 때, 독립표본의 모분산을 알 때와 모를 때, 합동분산을 가정할 수 있을 때와 없을 때, 대응표본, 모비율, 모분산 등 여러 케이스로 갈라진다. 하지만 전부 따로 외우기보다, 다음 구조를 먼저 잡고 상황에 맞게 분포와 표준오차를 바꾸는 것이 중요하다.

판단 순서 질문
1. 무엇을 추정하는가? 평균, 평균 차이, 비율, 분산 중 무엇인가?
2. 표본 구조는 무엇인가? 단일표본, 독립표본, 대응표본 중 무엇인가?
3. 모분산을 아는가? 알면 Z, 모르면 보통 t 또는 카이제곱으로 간다
4. 표준오차는 무엇인가? 해당 추정량이 표본마다 얼마나 흔들리는지 계산한다
5. 신뢰계수는 어디서 오는가? 정규분포표, t분포표, 카이제곱분포표 등에서 가져온다

공부 방식. 공식을 무조건 외우는 방식은 오래가기 어렵다. 바뀌는 조건을 보고 왜 Z에서 t로 바뀌는지, 왜 자유도가 생기는지, 왜 표준오차가 달라지는지를 논리적으로 추론해야 한다. 통계는 학문이면서 동시에 판단의 기술에 가깝다.

  • 신뢰구간의 기본 구조는 추정값 +- 신뢰계수 x 표준오차다.
  • 표준오차는 추정값의 표준편차이며, 표준오차를 줄이는 것은 중요한 성과다.
  • 기본 유의수준은 보통 5%, 즉 alpha = 0.05다.
  • 주요 Z 임계값은 99% = 2.58, 95% = 1.96, 90% = 1.645다.
  • 신뢰수준이 높을수록 신뢰구간의 폭은 넓어진다.
  • 두 모평균 차이는 독립표본과 대응표본을 먼저 구분해야 한다.
  • 모분산을 모르면 t분포와 자유도를 사용하고, 모분산 자체는 카이제곱분포와 연결된다.
  • 여러 케이스를 통째로 외우기보다 기본 구조에서 논리적으로 추론해야 한다.

Keywords: confidence interval, sampling error, standard error, confidence coefficient, significance level, Z critical value, t distribution, independent samples, paired samples, degrees of freedom, chi-square distribution, population variance, hypothesis testing

Week 12 centered on confidence intervals and the major inference cases that branch from them. The point was not to memorize many formulas separately, but to understand one basic structure and then substitute the correct standard error and confidence coefficient for each situation. The class connected one-sample means, two-sample mean differences, independent and paired samples, known and unknown variances, t distributions, and chi-square distributions.

Core idea. A target value is estimated by placing a sampling error around an estimate. Sampling error is the confidence coefficient multiplied by the standard error. This is why reducing standard error is a major achievement in statistical methods.

1. Basic Confidence Interval Form

The key structure of the week is the following two-line formula. For any parameter, start with an estimate and attach a sampling error around it.

target value = estimate +- sampling error
sampling error = confidence coefficient x standard error
Concept Meaning
Estimate A value computed from the sample, such as a sample mean, sample proportion, or difference between sample means
Standard error The standard deviation of an estimator across repeated samples
Confidence coefficient The critical value taken from the relevant distribution: normal, t, chi-square, and so on
Sampling error The half-width of a confidence interval, computed from the confidence coefficient and standard error

The "standard" in standard error refers to the standard deviation of an estimate. If we repeatedly sample from the same population, sample means or sample proportions will vary. The standard error measures that variation.

2. Significance Level and Confidence Interval Meaning

The significance level is usually written as alpha. If nothing is specified, 5%, or alpha = 0.05, is the default. The confidence level is 1 - alpha, so a 95% confidence interval corresponds to alpha = 0.05.

P(|error| <= c) = 1 - alpha
P(theta_hat - c <= theta <= theta_hat + c) = 1 - alpha
[theta_hat - c, theta_hat + c] = a 100(1 - alpha)% confidence interval for theta

As the confidence level increases, the confidence interval becomes wider. To include the parameter more reliably, we need a wider target interval. Therefore, confidence level and interval width should be read together.

3. Z Confidence Intervals and Critical Values

When the population variance or standard deviation is known and normal approximation is appropriate, we use critical values from the standard normal distribution. The middle probability is 1 - alpha, leaving alpha / 2 in each tail.

P(-z_(alpha/2) < Z < z_(alpha/2)) = 1 - alpha
Z = (X_bar - mu) / (sigma / sqrt(n))
mu = x_bar +- z_(alpha/2) x (sigma / sqrt(n))
Confidence level alpha z_(alpha/2) Note
99% 0.01 2.58 Widest interval among the three
95% 0.05 1.96 Common default
90% 0.10 1.645 Narrower interval

The empirical rule says that about 68%, 95%, and 99.7% of values fall within one, two, and three standard deviations of the mean. But for a central 95% interval under the standard normal distribution, the exact cutoff is about 1.96, not 2. Thus z_(alpha/2) is 1.96 for a 95% confidence interval.

4. Difference Between Two Means: Independent Samples

When comparing two population means, the estimate is the difference between two sample means. If the samples are independent and the two population variances are known, the distribution is:

(X_bar - Y_bar) ~ N(mu_1 - mu_2, sigma_1^2 / n_1 + sigma_2^2 / n_2)
(mu_1 - mu_2) = (x_bar - y_bar) +- z_(alpha/2) x sqrt(sigma_1^2 / n_1 + sigma_2^2 / n_2)

If the population variances are unknown, sample variances are used and the t distribution enters. If equal variances can be assumed, pooled variance is used. If not, Welch's degrees of freedom are computed, usually by software.

v = (s_1^2 / n_1 + s_2^2 / n_2)^2 / { (s_1^2 / n_1)^2 / (n_1 - 1) + (s_2^2 / n_2)^2 / (n_2 - 1) }
(mu_1 - mu_2) = (x_bar - y_bar) +- t_(alpha/2, v) x sqrt(s_1^2 / n_1 + s_2^2 / n_2)

5. Paired Samples

Paired samples are collected in pairs. Examples include the same person's resting heart rate and heart rate after a 100-meter sprint, or fasting blood sugar and blood sugar two hours after a meal.

Type Example Main treatment
Independent samples Comparing means from two separate groups Compute the standard error of the difference between two sample means
Paired samples Before-after comparison on the same subjects Create differences d_i and treat them like a one-sample mean problem
d_i = x_i - y_i
mu_D = d_bar +- t_(alpha/2, n-1) x s_D / sqrt(n)

The same data can be interpreted differently depending on the viewpoint. For example, in a midterm-final score comparison, students may feel that the midterm was harder, while an instructor may think students studied less for the final. Statistics is not only calculation; it also requires judgment about the comparison structure.

6. Chi-Square Distribution and Population Variance

Following the standard normal distribution means that a variable can be standardized as Z = (X - mu) / sigma. When the sample mean is used as the center, one degree of freedom is used up, and the squared sum follows a chi-square distribution.

sum((X_i - X_bar)^2 / sigma^2) ~ chi-square_(n-1)
sum(((X_i - X_bar) / sigma)^2) ~ chi-square_(n-1)

X_bar is not a parameter; it is a statistic used to estimate the population mean. Because the sample mean is used, the degrees of freedom become n - 1. Confidence intervals for population variance use critical values from the chi-square distribution.

7. Infer From Structure, Not Memorization

Confidence interval formulas branch into many cases: one mean, known or unknown variance, two independent means with known variances, unknown equal variances, unknown unequal variances, paired samples, proportions, and variances. Instead of memorizing them as isolated formulas, start from the following structure and change the distribution and standard error according to the case.

Step Question
1. What is being estimated? Mean, difference of means, proportion, or variance?
2. What is the sample structure? One sample, independent samples, or paired samples?
3. Is the population variance known? If known, use Z; if unknown, usually move to t or chi-square.
4. What is the standard error? Compute how much the estimator varies across samples.
5. Where does the confidence coefficient come from? Normal table, t table, chi-square table, and so on.

Study method. Blind memorization does not hold up well. The better approach is to infer why Z changes to t, why degrees of freedom appear, and why the standard error changes. Statistics is both a discipline and a craft of judgment.

  • The basic confidence interval form is estimate +- confidence coefficient x standard error.
  • Standard error is the standard deviation of an estimator, and reducing it is important.
  • The common default significance level is 5%, or alpha = 0.05.
  • Key Z critical values are 99% = 2.58, 95% = 1.96, and 90% = 1.645.
  • Higher confidence levels produce wider confidence intervals.
  • For two mean differences, first distinguish independent samples from paired samples.
  • When population variance is unknown, t distributions and degrees of freedom matter; population variance connects to chi-square.
  • Do not memorize every case separately; infer from the basic structure.
